# Workflow

这个目录专门用于讨论和沉淀“信息收集类 Playwright 项目”的工作流设计。

目录结构遵循当前项目约定：

- `main.py` 作为总调用文件
- `modules/` 存放被调用的工作流模块
- `*.md` 存放工作流规则说明

当前目标：

- 总结这类项目的通用工作流
- 抽象可复用的模块生成顺序
- 设计参数驱动的 AI Agent 流水线
- 记录默认值、降级策略和异常处理策略

建议后续把这类内容逐步放在这里：

- 工作流总览
- 模块清单
- 参数协议
- 默认值策略
- 生成顺序
- 测试策略
- 交付模板

后续如果继续细化，可以在这个目录下新增：

- `pipeline.md`
- `module_contracts.md`
- `defaults.md`
- `examples/`
