# business_risk 模块说明

## 文件夹结构

```
business_risk/
├── __init__.py         # 模块初始化文件
├── business_risk_main.py  # 测试主脚本
├── navigator.py        # 导航模块
├── tag_nav_extractor.py   # 标签导航提取模块
├── date_range_filter.py   # 日期范围筛选模块
├── vip_detector.py     # VIP检测模块
└── README.md           # 本说明文件
```

## 文件作用说明

### 1. business_risk_main.py（测试主脚本）

**功能**：经营风险分析主模块，负责协调各个子模块的调用，实现完整的经营风险分析流程。

**流程图**：

```
┌─────────────────────┐
│ 开始处理经营风险分析 │
└──────────┬──────────┘
           ↓
┌─────────────────────┐
│ 循环处理每个公司名称 │
└──────────┬──────────┘
           ↓
┌─────────────────────┐     ┌─────────────────────┐
│ 进入公司详情页      │────>│ 点击经营风险标签    │
└──────────┬──────────┘     └──────────┬──────────┘
           ↓                          ↓
┌─────────────────────┐     ┌─────────────────────┐
│ 提取标签导航文本    │<────│ 提取经营风险详情数据 │
└──────────┬──────────┘     └──────────┬──────────┘
           ↓                          ↓
┌─────────────────────┐     ┌─────────────────────┐
│ 回到首页            │<────│ 输出分析结果        │
└──────────┬──────────┘     └─────────────────────┘
           ↓
┌─────────────────────┐
│ 经营风险分析处理完成 │
└─────────────────────┘
```

### 2. navigator.py（导航模块）

**输入**：Playwright Page 对象

**输出**：None（无返回值）

**业务逻辑**：
1. 查找包含文本"经营风险"的按钮（非严格模式）
2. 点击找到的第一个匹配元素
3. 处理可能的异常情况

### 3. tag_nav_extractor.py（标签导航提取模块）

**输入**：Playwright Page 对象

**输出**：二元数组，每个元素为(文本, 数字)，例如 ["注销备案", "0"]

**业务逻辑**：
1. 找到id为"JS_tag_nav"的元素
2. 获取所有子元素
3. 对每个子元素提取非数字部分和数字部分
4. 将它们组成二元组存储到结果数组中

### 4. date_range_filter.py（日期范围筛选模块）

**输入**：
- Playwright Page 对象
- 日期范围起始（格式：YYYY-MM-DD）
- 日期范围结束（格式：YYYY-MM-DD）

**输出**：列表，每个元素对应一个有数据的板块，包含板块标题、data-dim值和符合日期范围的行数据

**业务逻辑**：
1. 自动发现页面上所有带有data-dim属性的元素
2. 对每个元素执行以下操作：
   - 获取data-dim属性值
   - 提取板块标题
   - 调用VIP检测模块判断是否为VIP板块
   - 如果是VIP板块，跳过
   - 如果不是VIP板块，提取表格数据
   - 定位日期列（优先从表头查找，降级从数据行查找）
   - 遍历所有数据行，筛选符合日期范围的行
3. 返回符合要求的板块数据列表

### 5. vip_detector.py（VIP检测模块）

**输入**：
- Playwright Page 对象
- 业务标识符（如 "corpIllegals"）

**输出**：布尔值，如果是VIP板块返回True，否则返回False

**业务逻辑**：
1. 查找指定data-dim的板块容器
2. 提取板块标题（用于日志）
3. 提取板块容器内的全部文本并转换为小写
4. 统计文本中包含的VIP关键字数量
5. 如果关键字数量≥2，判定为VIP板块，输出警告日志并返回True
6. 否则返回False
